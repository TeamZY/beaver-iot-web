# @smartoffice/shared

跨端/跨项目共享的工具、组件、服务、类型等集合。

## 目录结构

```
.
├── src
│   ├── components      # 通用组件
│   ├── config          # 通用配置
│   ├── hooks           # 通用 Hook
│   ├── services        # 通用服务
│   └── utils           # 通用工具
├── types               # 通用类型
├── README.md
├── package.json
└── tsconfig.json
```
